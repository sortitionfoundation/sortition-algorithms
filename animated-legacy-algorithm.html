How it works

  1. Each round, compute difficulty = (min − selected) / remaining for every feature value
  2. Pick a random person from the value with the highest difficulty
  3. Update the selected and remaining numbers for all values that person has
  4. When a value hits its max, remove all remaining candidates with that value from the pool
  5. Update the remaining numbers for all values those removed people had
